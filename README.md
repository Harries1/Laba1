# 💻 Hardware Monitor (WPF + WMI)

Приложение для мониторинга аппаратных ресурсов компьютера, разработанное на **C# (.NET, WPF)** с использованием технологии **Windows Management Instrumentation (WMI)**.

Программа позволяет получать системную информацию и экспортировать отчёты в различные форматы.


## 📌 Возможности

Приложение предоставляет информацию о:

* 🧠 **CPU**

  * Модель процессора
  * Количество ядер и потоков
  * Тактовая частота
  * Текущая загрузка

* 🧩 **RAM**

  * Общий объём памяти
  * Свободная память
  * Использование в процентах
  * Информация о планках (производитель, объём, частота)

* 💾 **Диски**

  * Физические диски (модель, размер)
  * Логические диски (размер, свободное место, файловая система)

* 🎮 **Видеоадаптер**

  * Название
  * Объём видеопамяти
  * Разрешение экрана

* 🌐 **Сетевые адаптеры**

  * Название
  * MAC-адрес
  * Скорость
  * Статус подключения

* 🖥 **Система**

  * Версия Windows
  * Архитектура ОС
  * Имя компьютера
  * Пользователь

## 📤 Экспорт отчёта

Поддерживаются форматы:

* `TXT`
* `CSV`
* `JSON`

Файлы создаются в директории запуска приложения.


## 🏗 Архитектура проекта

Проект реализован с использованием принципов **MVVM**:

* **Model** — модели данных оборудования
* **WmiService** — сервис для получения данных через WMI
* **ViewModel** — логика представления
* **View (XAML)** — пользовательский интерфейс

Используется асинхронное обновление данных (`async/await`) для предотвращения блокировки интерфейса.


## 🖼 Интерфейс

Интерфейс построен на WPF и включает:

* Панель управления (обновление + экспорт)
* Вкладки:

  * CPU
  * RAM
  * Диски
  * Видео
  * Сеть
  * Система
